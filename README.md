# Tic-Tac-Toe C++

Um Jogo da Velha (_Tic-Tac-Toe_) desenvolvido em C++ para rodar no terminal. O projeto foi estruturado utilizando os pilares da Programação Orientada a Objetos (POO) e o CMake foi utilizado como sistema de automação de build.

![C++](https://img.shields.io/badge/C%2B%2B-blue.svg)
![CMake](https://img.shields.io/badge/CMake-green.svg)

---

## 🛠️ Tecnologias e Ferramentas

- **Linguagem:** C++
- **Sistema de Build:** CMake

---

## 🚀 Como Executar o Projeto

Pré-requisitos

Antes de começar, certifique-se de ter instalado:

- Compilador C++ suportando o padrão C++20 (como o g++ ou clang).
- Utilitário CMake.
- Gerenciador de automação make.

Passo a Passo para Compilação

1. Clone este repositório em sua máquina local:

   ```bash
    git clone https://github.com/danilocamposdev/tic-tac-toe.git
    cd to-do-list
   ```

2. Compile o código fonte gerando o executável:

   ```bash
    cmake --build build
   ```

3. Execute a aplicação:

   ```bash
    ./build/to-do-list
   ```
